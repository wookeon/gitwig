# Gitwig

[English](./README.md) | 한국어

**브랜치가 많은 작업자를 위한 macOS Git 클라이언트.**

Gitwig는 저장소 히스토리, 작업 중인 변경사항, 브랜치, 리모트, 프로필, GitHub 계정 상태를 데스크톱 작업 공간 한곳에 모읍니다.

Gitwig는 GitHub Releases에서 무료로 배포하는 비오픈소스 소프트웨어입니다. 현재 공개 버전은 `0.0.2`입니다.

[다운로드](https://github.com/wookeon/gitwig/releases) | [릴리스 노트](./changelog/README.ko.md) | [이슈 제보](https://github.com/wookeon/gitwig/issues) | [오픈소스 고지](./THIRD_PARTY_NOTICES.md)

## Gitwig로 할 수 있는 일

- commit history, branch 흐름, working changes, staged changes를 함께 확인합니다.
- 파일 diff, content, history, blame, conflict resolution 화면을 확인합니다.
- 파일과 hunk를 stage/unstage하고, stash, discard, Finder에서 보기를 실행합니다.
- checkout, merge, rebase, fetch, pull, push, upstream 설정 같은 브랜치와 리모트 작업을 처리합니다.
- 프로필로 개인 작업과 업무 작업의 작성자 정보를 분리합니다.

## 다운로드

Gitwig는 GitHub Releases에서 배포합니다.

- 다운로드: [GitHub Releases](https://github.com/wookeon/gitwig/releases)
- 현재 공개 버전: `0.0.2`
- 요구사항: macOS 26.0 이상

## 피드백과 이슈 제보

요구사항, 기능 요청, 개선 제안, 버그 제보, 호환성 질문은 [GitHub Issues](https://github.com/wookeon/gitwig/issues)에 남겨주세요.

버그를 제보할 때는 Gitwig 버전, macOS 버전, 문제가 발생한 Git 작업, 재현 절차, 기대한 결과, 실제 결과를 함께 적어주세요. 접근 토큰, 비공개 커밋 내용, 고객 정보, 민감한 로그는 공개 이슈에 올리지 마세요.

## 라이선스와 개인정보

Gitwig는 무료로 사용할 수 있는 비오픈소스 소프트웨어입니다. 이 저장소는 소스 코드를 배포하지 않습니다. GitHub 계정 연결은 OAuth device flow를 사용하며, 인증 정보는 macOS Keychain에 저장됩니다. 자세한 내용은 [`LICENSE`](./LICENSE)와 [`THIRD_PARTY_NOTICES.md`](./THIRD_PARTY_NOTICES.md)를 확인하세요.
