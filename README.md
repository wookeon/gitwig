# Gitwig

English | [한국어](./README.ko.md)

**A focused macOS Git client for people who live in branches.**

Gitwig brings repository history, working changes, branches, remotes, profiles, and GitHub account state into one desktop workspace.

Gitwig is freeware, not open source. The current public version is `0.0.2`.

[Download](https://github.com/wookeon/gitwig/releases) | [Release Notes](./changelog) | [Issues](https://github.com/wookeon/gitwig/issues) | [Third-Party Notices](./THIRD_PARTY_NOTICES.md)

## What Gitwig Helps With

- Review commit history, branch flow, working changes, and staged changes together.
- Inspect file diffs, content, history, blame, and conflict resolution views.
- Stage or unstage files and hunks, stash changes, discard changes, and reveal files in Finder.
- Run common branch and remote workflows, including checkout, merge, rebase, fetch, pull, push, and upstream setup.
- Keep personal and work author identities separate with profiles.
