# Gitwig 0.0.2

[English](./v0.0.2.md) | 한국어

Gitwig `0.0.2`는 터미널 패널을 추가하고, 브랜치 정리, 태그 생성, 많은 변경사항 검토, 업데이트 안내를 더 수월하게 만든 릴리스입니다.

## 새 기능

- 사이드바에서 Shift 또는 Command로 여러 브랜치를 선택해 삭제 가능한 local 또는 remote 브랜치를 한 번에 삭제할 수 있습니다.
- 히스토리에서 annotated tag를 만들고, 선택한 remote로 바로 push할 수 있습니다.
- SwiftTerm 기반 터미널 패널을 열어 Gitwig 안에서 명령줄 작업을 이어갈 수 있으며, D2Coding 글꼴을 함께 제공합니다.

## 변경

- unstaged, staged, commit 영역의 높이를 조정해 변경 파일이 많아도 더 편하게 검토할 수 있습니다.
- 작업 변경사항 목록에서 여러 파일을 선택해 stage, unstage, stash, discard, 경로 복사, 파일명 복사, Finder에서 보기를 실행할 수 있습니다.
- 지금 보고 있는 히스토리 범위 밖의 태그도 사이드바에서 더 안정적으로 확인할 수 있습니다.
- 새 버전이 있거나 업데이트가 필요할 때 더 분명한 안내를 볼 수 있습니다.

## 수정

- commit 또는 stash의 변경 파일 목록이 길어도 이동 중 선택한 파일 행이 화면 안에 유지됩니다.
- merge summary 행이 첫 선택 클릭에 바로 펼쳐지지 않아 히스토리를 둘러보다가 의도치 않게 열리는 일이 줄었습니다.
- 사이드바, 히스토리, 파일 브라우저, 작업 변경사항 사이를 이동할 때 오래된 선택 하이라이트가 더 일관되게 정리됩니다.
- 파일 내용, 히스토리, blame 정보가 없을 때 상세 화면의 빈 상태가 더 간결하게 표시됩니다.
