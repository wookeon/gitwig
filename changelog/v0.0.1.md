# Gitwig 0.0.1

English | [한국어](./v0.0.1.ko.md)

**Initial public release.**

Gitwig `0.0.1` is the first public release of Gitwig, a freeware macOS Git client distributed through GitHub Releases.

## Highlights

- First public build of Gitwig for macOS.
- Repository history, working changes, staging, branches, remotes, stashes, tags, submodules, and profiles are available in one desktop workspace.
- GitHub account connection is supported first.

## Notes

- Requires macOS 26.0 or later.
- Automatic updates are not available yet.
- Gitwig is freeware, not open source.
