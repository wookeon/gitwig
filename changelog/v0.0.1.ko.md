# Gitwig 0.0.1

[English](./v0.0.1.md) | 한국어

**첫 공개 릴리스입니다.**

Gitwig `0.0.1`은 GitHub Releases를 통해 무료 배포 소프트웨어로 제공되는 macOS용 Git 클라이언트의 첫 공개 릴리스입니다.

## 핵심

- macOS용 Gitwig 첫 공개 빌드입니다.
- 저장소 히스토리, 변경사항, 스테이징, 브랜치, 리모트, stash, tag, submodule, 프로필 기능을 하나의 데스크톱 작업 공간에서 사용할 수 있습니다.
- GitHub 계정 연결을 먼저 지원합니다.

## 알아둘 점

- macOS 26.0 이상이 필요합니다.
- 자동 업데이트는 아직 제공하지 않습니다.
- Gitwig는 오픈소스가 아닌 무료 배포 소프트웨어입니다.
